# Plataforma ONG — Entregas AE1–AE4

Projeto completo com versionamento, acessibilidade, CSS/JS modular e estrutura pronta para deploy.

## Estrutura
- AE1: HTML + imagens otimizadas (JPG/WebP)
- AE2: + CSS modular (base/layout/components/utilities/theme)
- AE3: + JS modular (SPA básica + validação de formulário + storage)
- AE4: + package.json (build), documentação completa

## Como rodar
Abra `index.html` localmente. Para SPA funcionar, use `#/home`, `#/projetos`, `#/cadastro`.

## Build
`npm install` e `npm run build`.
